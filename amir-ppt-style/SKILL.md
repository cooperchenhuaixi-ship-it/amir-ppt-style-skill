# Amir PPT Style

## Purpose

Use this skill to create or restyle presentations with the learned Amir deck language: oversized editorial typography, strong page-by-page color systems, premium tech gradients, compact narrative pacing, and sparse but sharp commercial storytelling.

Do not copy the original decks or reuse their screenshots unless the user explicitly provides permission for that specific output. Treat the source decks as style references, not as assets to redistribute.

## Workflow

1. Use the presentation-building workflow required by the active deck tool or skill.
2. Read `references/style-system.md` before designing slide structure, choosing colors, or writing layout code.
3. Choose one primary route:
   - `dark-tech`: deep black, blue or violet neon, glassy data blocks, product/investor/campaign energy.
   - `bold-proposal`: one vivid flat background per slide, oversized editorial titles, proposal/campaign energy.
   - `soft-media`: white or black editorial canvas, pastel glow gradients, creator/media kit energy.
4. Build a short, paced deck rather than a dense report. Prefer 6-10 slides unless the user requests otherwise.
5. Make each slide answer one job: cover, objective, insight, mechanism, proof, timeline, economics, ask, or closing line.
6. Verify visually that text does not wrap awkwardly, huge titles remain intentional, and translucent panels do not fight the background.

## Design Rules

- Use giant title typography as the main visual object: usually 72-180pt on 16:9 slides.
- Pair huge titles with tiny section labels: 14-18pt, top-left, often numbered like `04 · CHANNEL MIX`.
- Use modern geometric fonts when available: Geist, Inter, DM Sans, Space Grotesk, or close equivalents.
- Avoid generic corporate template styling. Do not use default title placeholders, stock icon grids, or heavy boxed layouts.
- Let one visual motif carry the deck: fluid neon ribbons, dot matrices, blurred glow, large cropped typography, or bold flat color.
- Keep body copy short. If a slide needs paragraphs, split the thought across slides.
- Use cards as supporting instruments only: translucent, quiet, and secondary to the main headline or metric.
- Close with a large direct line, not a thank-you slide.
